Botón cuadrado de solo icono para barras de herramientas y acciones de fila.

```jsx
<IconButton aria-label="Filtrar"><i data-lucide="sliders-horizontal" /></IconButton>
<IconButton variant="ghost" size="sm"><i data-lucide="more-vertical" /></IconButton>
```

Variantes `default | ghost`, tamaños `sm | md`, `active` para estado seleccionado. Siempre incluye `aria-label`.
