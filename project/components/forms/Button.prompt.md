Botón de acción de TankFlow; úsalo para acciones primarias, secundarias y destructivas en formularios, barras de herramientas y diálogos.

```jsx
<Button variant="primary" iconLeft={<i data-lucide="log-in" />}>Registrar ingreso</Button>
<Button variant="secondary" size="sm">Cancelar</Button>
<Button variant="danger">Marcar crítico</Button>
```

Variantes: `primary` (azul señal), `secondary` (superficie + borde), `ghost`, `danger`, `success`. Tamaños: `sm | md | lg`. Usa `block` para ancho completo y `iconLeft` / `iconRight` con iconos Lucide.
