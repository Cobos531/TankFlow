Menú desplegable nativo con estilo TankFlow.

```jsx
<Select label="Tipo de reparación"
  placeholder="Seleccionar…"
  options={['Estructural','Interior','Rotulación','Pintura','Inspección regulatoria']} />
```

Acepta `options` como strings o `{value,label}`. Para casos avanzados pasa `<option>` como children.
