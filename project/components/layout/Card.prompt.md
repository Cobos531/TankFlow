Contenedor de superficie con cabecera y acciones opcionales.

```jsx
<Card title="Equipos en alerta" actions={<Button variant="ghost" size="sm">Ver todos</Button>}>
  …contenido…
</Card>
<Card bodyFlush>{/* tabla a sangre */}</Card>
```

Usa `bodyFlush` para tablas, `raised` para superficie elevada con sombra.
